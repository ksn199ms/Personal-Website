Food Delivery Web Application Using MERN Stack

TastyTrack is a food delivery web application built using the MERN stack. It allows users to browse and filter restaurant menus, add items to their cart, make secure payments via Razorpay, and track their order status in real time. Admins can manage food items, view customer orders, and update order statuses, providing seamless restaurant management.
